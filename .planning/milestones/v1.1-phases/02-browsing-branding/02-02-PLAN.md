  <files>src/js/main.js, src/css/main.css</files>
  <action>
**Replace `src/js/main.js`** with a proper ES module implementing all interactive behaviors:

1. **initCopyLinks():** Query all `[data-copy-link]` buttons. On click:
   - Prevent default
   - Build URL: `window.location.origin + window.location.pathname + '#' + btn.dataset.copyLink`
   - Use `navigator.clipboard.writeText(url)` with try/catch
   - Fallback: create temporary input, select, execCommand('copy'), remove
   - Show tooltip: create `<span>` with "Link kopiert!" text, position absolute above button (-top-8, centered), bg-gruene-dark text-white text-xs px-2 py-1 rounded
   - Remove tooltip after 2000ms via setTimeout
   - On mobile (no hover): copy buttons should always be visible (Plan 01 handles this via CSS `group-hover` + `sm:opacity-0` pattern; if not, add note)

2. **initScrollToTop():** Find `#scroll-to-top` button.
   - On scroll (passive listener): toggle `hidden` class based on `window.scrollY > 300`
   - On click: `window.scrollTo({ top: 0, behavior: 'smooth' })`

3. **initBundeslandDropdown():** Find `#bundesland-nav` select.
   - On change: if value is not empty, `window.location.href = e.target.value`
   - Use relative paths (the select options contain relative paths like `../gemeindeordnungen/wien.html`)

4. **DOMContentLoaded listener** that calls all three init functions.

**Extend `src/css/main.css`** (if not already done in Plan 01, add these):

1. **Anchor highlight animation:**
```css
article:target {
  animation: anchor-highlight 2s ease-out;
  border-radius: 0.25rem;
}
@keyframes anchor-highlight {
  0% { background-color: var(--color-gruene-light); }
  100% { background-color: transparent; }
}
```

2. **Copy button visibility on hover (desktop) / always visible on mobile:**
```css
.copy-link-btn {
  transition: opacity 0.15s;
}
@media (hover: hover) {
  .copy-link-btn {
    opacity: 0;
  }
  article:hover .copy-link-btn {
    opacity: 1;
  }
}
```

3. **Scroll-to-top button styling:**
```css
#scroll-to-top {
  transition: opacity 0.2s;
}
```
