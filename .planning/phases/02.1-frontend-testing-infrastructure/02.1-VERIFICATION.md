## Notable Deviation: baseURL includes /src/ path

The plan specified `baseURL: 'http://localhost:4173/gemeindeordnung'` but implementation uses `http://localhost:4173/gemeindeordnung/src/`. This is correct because Vite's multi-page build with input files in `src/` outputs HTML to `dist/src/`, making pages served at `/gemeindeordnung/src/`. The fix was properly identified and documented in SUMMARY.md. All test goto paths (`'./gemeindeordnungen/wien.html'`, `'./index.html'`) resolve correctly against this baseURL and confirmed by the presence of 15 screenshot files showing tests actually executed.
