2. **Add `injectStructuralMarkers()` function** -- new function placed near `injectGlossaryTerms()`:
   ```javascript
   function injectStructuralMarkers(html) {
     if (!html) return html;
     // Match legal reference patterns in text content (between > and <)
     // Process the HTML by splitting on tags to only modify text nodes
     return html.replace(/>[^<]+</g, (match) => {
       let text = match.slice(1, -1); // Remove > and <
       // Abs. N references
       text = text.replace(/\b(Abs\.\s*\d+[a-z]?)/g, '<span class="legal-ref">$1</span>');
       // Paragraph sign references
       text = text.replace(/(§\s*\d+[a-z]?(?:\s*(?:bis|und|iVm)\s*§?\s*\d+[a-z]?)*)/g, '<span class="legal-ref">$1</span>');
       // Z (Ziffer) references
       text = text.replace(/\b(Z\s*\d+)/g, '<span class="legal-ref">$1</span>');
       // lit. references
       text = text.replace(/\b(lit\.\s*[a-z])/g, '<span class="legal-ref">$1</span>');
       return '>' + text + '<';
     });
   }
   ```

3. **Call `injectStructuralMarkers()` on the body** in `renderParagraph()` -- call it BEFORE `injectGlossaryTerms()` to avoid conflicts with glossary tooltip HTML:
   ```javascript
   // Inject structural markers BEFORE glossary tooltips
   if (body) {
     body = injectStructuralMarkers(body);
   }
   // Inject glossary tooltips into body text ONLY (not summary)
   if (glossaryTerms && glossaryTerms.length > 0 && body) {
     body = injectGlossaryTerms(body, glossaryTerms, '../');
   }
   ```
   Move the structural markers injection to be BEFORE glossary injection (currently glossary is at line 114-116). This ordering is important: structural markers wrap small text fragments in `<span>`, then glossary injection works on the resulting text without issue.
