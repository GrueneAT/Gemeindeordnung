<objective>
Improve law text typography, section heading hierarchy, and Absatz separation on all law pages.

Purpose: Law pages currently present legal text as a wall of text with minimal visual hierarchy. This plan adds generous typography (17px base, 1.7+ line-height, ~65ch max-width), strong visual differentiation between Hauptstueck/Abschnitt/Paragraph headings, and clear separation of numbered Absaetze within paragraphs -- making the site feel like a well-typeset legal commentary.

Output: Updated CSS styles, updated HTML template generation, updated/new E2E tests.
</objective>

3. **`.hauptstueck-heading`** -- Hauptstueck headings:
   - `font-size: 1.5rem`, `font-weight: 700`, `margin-top: 3rem`, `margin-bottom: 1.5rem`
   - `padding-top: 1.5rem`, `padding-bottom: 0.75rem`
   - `border-top: 3px solid var(--color-gruene-green)`, `color: var(--color-gruene-dark)`

4. **`.abschnitt-heading`** -- Abschnitt headings:
   - `font-size: 1.25rem`, `font-weight: 600`, `margin-top: 2rem`, `margin-bottom: 1rem`
   - `padding-left: 0.75rem`, `border-left: 3px solid var(--color-gruene-green)`

5. **`.law-text h3`** -- Paragraph headings subordinate styling:
   - `font-size: 1.0625rem`, `font-weight: 600`

6. **`h2[id]`** -- add `scroll-margin-top: 5rem` (match existing `h3[id]` rule)

7. **Absatz styles** -- `.absaetze-container`, `.absatz`, `.absatz-num`, `.absatz-text`:
   - `.absatz`: `display: flex`, `gap: 0.5rem`, `padding: 0.5rem 0`, `border-bottom: 1px solid rgba(0,0,0,0.05)`
   - `.absatz:last-child`: no border-bottom
   - `.absatz-num`: `opacity: 0.5`, `font-size: 0.875rem`, `flex-shrink: 0`, `min-width: 2rem`, `font-variant-numeric: tabular-nums`
   - `.absatz-text`: `flex: 1`

**Template changes in `scripts/generate-pages.js`**:

1. **`renderSection()`** (lines 148-169): Replace inline Tailwind classes on h2 headings with semantic CSS classes:
   - Hauptstueck: `class="hauptstueck-heading"` (remove old `text-2xl font-bold mt-12 mb-6 text-gruene-dark`)
   - Abschnitt: `class="abschnitt-heading"` (remove old `text-xl font-semibold mt-8 mb-4 text-gruene-dark`)
   - Keep the `id="${sectionSlug}"` attribute
   - Update section wrapper: replace border Tailwind classes with `.hauptstueck-section` / `.abschnitt-section` if needed, or keep simple

2. **`renderParagraph()`** (lines 106-108): Replace `<ol><li>` Absatz rendering with separated `<div>` blocks:
   ```javascript
   if (para.absaetze && para.absaetze.length > 0) {
     const items = para.absaetze.map(a => {
       const numLabel = a.nummer ? `<span class="absatz-num">(${a.nummer})</span>` : '';
       // Strip leading "(N)" from text since we render the number separately
       const cleanText = a.nummer ? a.text.replace(new RegExp(`^\\(${a.nummer.replace(/[.*+?^${}()|[\]\\]/g, '\\$&')}\\)\\s*`), '') : a.text;
       return `<div class="absatz">${numLabel}<span class="absatz-text">${escapeHtml(cleanText)}</span></div>`;
     }).join('\n');
     body = `<div class="absaetze-container">\n${items}\n</div>`;
   }
   ```
   IMPORTANT: Only strip leading `(N)` if `a.nummer` exists AND text starts with that pattern. Use regex-escaped `a.nummer` to handle potential `1a` style numbers.

3. **Law page `<main>` tag** (line 451): Add `law-text` class:
   ```html
   <main data-pagefind-body class="law-text max-w-prose mx-auto leading-relaxed">
   ```
