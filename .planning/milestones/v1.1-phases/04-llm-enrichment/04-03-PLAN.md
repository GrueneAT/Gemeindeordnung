**1. Glossary tooltip injection in generate-pages.js:**

Add `injectGlossaryTerms(bodyHtml, glossaryTerms, categoryPrefix)` function:
- Takes the paragraph **body text HTML only** (NOT the full article HTML, NOT the summaryHtml from Plan 02's `<details>` block)
- CRITICAL: In `renderParagraph()`, Plan 02 added a `<details>` block with `summaryHtml` for the "Vereinfachte Zusammenfassung". The `injectGlossaryTerms()` function must be called ONLY on the body text variable (the rendered paragraph text/absaetze content), BEFORE it is assembled with the summary block into the final `<article>`. This ensures glossary tooltips appear only in the legal text, not inside summary descriptions.
- For each term, matches whole words only (first occurrence per paragraph only)
- Wraps match in tooltip markup:
  ```html
  <span class="glossar-term" tabindex="0">{matched text}<span class="glossar-tooltip">{definition}<a href="{prefix}glossar.html#{slug}" class="glossar-tooltip-link">&#8594; Glossar</a></span></span>
  ```
- Be careful NOT to match inside HTML tags or attributes
- Use conservative matching: `new RegExp('\\b(' + escapeRegex(term.term) + ')\\b', 'i')` for first match only
- Apply AFTER `escapeHtml()` on the paragraph text content (match against the already-escaped text)
- `categoryPrefix` is `../` for law pages (they are in subdirs)

Load glossary terms in `generatePages()` and pass to `renderParagraph()`:

```javascript
// 1. Render the body HTML from para.text / para.absaetze (existing logic)
```

**4. Add mobile tooltip JS in main.js:**

Add `initGlossaryTooltips()` function for mobile tap support:
```javascript
function initGlossaryTooltips() {
  // On touch devices, toggle tooltip on tap
  document.addEventListener('click', (e) => {
    const term = e.target.closest('.glossar-term');

    // Close all open tooltips first
    document.querySelectorAll('.glossar-tooltip-active').forEach(el => {
      if (el !== term) el.classList.remove('glossar-tooltip-active');
    });

    if (term) {
      e.preventDefault();
      term.classList.toggle('glossar-tooltip-active');
    }
  });
}
```
